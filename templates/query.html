Sample Queries
Basic Selection
SELECT * FROM students Get all students
Filter by Grade
SELECT * FROM students WHERE avg_grade > 80 Get students with average grade above 80
Search by Name
SELECT * FROM students WHERE name LIKE 'John' Find students with 'John' in their name
Grade Range
SELECT * FROM students WHERE avg_grade >= 70 AND avg_grade <= 90 Students with grades between 70 and 90
Sort by Grade
SELECT * FROM students ORDER BY avg_grade DESC Sort students by grade (highest first)
Limited Results
SELECT * FROM students ORDER BY name ASC LIMIT 5 Get first 5 students alphabetically
Course Filter
SELECT name FROM students WHERE courses LIKE 'Math' Students enrolled in Math course
Group by Course
GROUP BY courses Group students by their courses
Multiple Courses
SELECT * FROM students WHERE course_count > 2 Students enrolled in more than 2 courses
Quick Reference
Available Fields:
  • roll_no - Student roll number
  • name - Student name
  • email - Student email
  • courses - Enrolled courses
Computed Fields:
  • grades - All grades
  • avg_grade - Average grade
  • course_count - Number of courses
Operators:
=, !=, >, <, >=, <=, LIKE, IN
Logical:
AND, OR
